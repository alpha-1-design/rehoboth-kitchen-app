# Presentation Generator

AI-powered PWA that transforms text, objects, faces, and products into beautiful presentations.

## Quick Start

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Features

- 📄 Text to PowerPoint
- 🔍 Object Detection
- 👤 Face Recognition
- 🛍️ Product Search
- 🌍 Location-based customization
- 🌐 Multi-language support
- 🎨 Dark/Light mode
- ✨ Premium animations
- 📱 PWA (installable)
- 🔗 QR code sharing

## Tech Stack

- React.js
- Tailwind CSS
- Framer Motion
- TensorFlow.js
- Tesseract.js
- pptxgen-js
